# Cgo stack
when use cgo, how to print stack for both c and go

## How to build and run

    $ git clone git@github.com:xiaofan-luan/cgo_stack.git
    $ cd cgo_stack
    $ mkdir build
    $ cd build
    $ cmake ../
    $ make
    $ ./example/test

Thanks https://github.com/mutse/go-cmake for the demo of cmake go build
